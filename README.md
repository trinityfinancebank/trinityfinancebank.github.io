# Trinity Finance Bank — Demo Dashboard

This is a static demo / mockup of a banking dashboard (demo only). It reproduces the layout and data from a provided screenshot and includes:
- Account cards (balance, account type, total transactions, account number)
- Transaction history table with sample transactions
- Search/filter, add demo transaction, and CSV download features

Important: This is a mock/demo site. Do not collect real credentials or real funds. Not a real bank.

How to run locally:
1. Clone the repo:
   git clone https://github.com/trinityfinancebank/trinityfinancebank.github.io.git
2. Open index.html in a browser or serve with a simple static server.
